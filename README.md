# Rainy Session 2083 – ESP32 & Microcontroller Workshop

This repository contains the code, examples, and activity solutions from the **Rainy Session 2083 Robotics Beginner Workshop**.

During the 7-day workshop, I served as a tutor for two sessions focused on **Microcontrollers, ESP32, and Embedded Programming**. The repository was created to help participants review concepts, practice independently, and access all workshop materials in one place.

## Prerequisites

Before running the projects, ensure you have the following:

### Hardware

* ESP32 Development Board
* USB Cable
* Breadboard
* LEDs
* Resistors
* Push Buttons
* Sensors 
* Motors 

### Software

* Arduino IDE or PlatformIO
* ESP32 Board Package installed
* USB Drivers for your ESP32 board

## Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/<your-username>/<repository-name>.git
```

### 2. Open a Project

Navigate to the desired activity folder and open the corresponding project in Arduino IDE or PlatformIO.

### 3. Select Your Board

For Arduino IDE:

```text
Tools → Board → ESP32 Arduino → ESP32 Dev Module
```

### 4. Upload the Code

1. Connect the ESP32 board.
2. Select the correct COM/Serial port.
3. Upload the program.


Feel free to explore, modify, and build upon these examples as you continue your robotics and embedded systems learning journey.
